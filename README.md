## 搬瓦工官网登录步骤详解

### 第一步：进入客户中心（Client Area）

1. 打开镜像站地址，比如 bwh81.net
2. 右上角点 **"Client Area"**
3. 输入注册时的邮箱和密码
4. 登录完成，进入账户主界面

进来之后能看到账户余额、账单记录、工单系统、账号设置这些。你买的 VPS 在哪？点顶部菜单 **Services → My Services**，所有已购买的 VPS 都在这里列出来。

### 第二步：进入 KiwiVM 管理面板

找到对应的 VPS，点击进去，左侧菜单有一个 **"KiwiVM Control Panel"** 按钮，点击之后会自动跳转到这台 VPS 专属的管理界面。

不需要额外输密码。这个跳转是自动登录的，相当方便。

### 第三步：在 KiwiVM 里能干什么

进到 KiwiVM 之后，左侧是功能菜单，右侧是 VPS 实时状态。常用功能包括：

1. **Main controls** — 查看运行状态，执行重启、关机操作
2. **Install new OS** — 一键重装操作系统（CentOS、Debian、Ubuntu、Rocky Linux 等）
3. **Root password modification** — 忘记 root 密码在这里重置
4. **Snapshots** — 手动给 VPS 创建快照备份
5. **Migrate to another DC** — 一键切换机房，不丢数据
6. **Two-factor authentication** — 给 KiwiVM 面板开启两步验证

KiwiVM 是搬瓦工自研的管理面板，同价位里能做到这个程度的不多——特别是机房一键迁移这个功能，很多同行要么没有，要么得提工单等客服手动操作。

## 搬瓦工官网打不开、登录失败怎么处理

这几个问题出现频率很高，逐一说一下：

**问题一：镜像站也打不开**

先确认你有没有开着代理或梯子。搬瓦工在注册购买阶段要求关闭代理，否则系统可能判定 IP 异常，轻则无法付款，重则订单被取消自动退款。关掉代理，直接访问镜像站地址。

**问题二：登录后台时提示"Checking your browser"**

这是 Cloudflare 的安全验证，刷新一下就过了。偶尔出现，不是报错。

**问题三：忘记密码**

客户中心登录页面有"Forgot Password"，填注册邮箱发重置链接。邮件可能落进垃圾邮件箱，先去查一下。

**问题四：购买时提示欺诈订单（Fraud Order）**

多数情况是因为下单时开着代理，IP 地址和填写的国家信息对不上。关掉代理重新下单，国家信息如实填写（可以是中国），地址写拼音也行，但邮箱和密码要用自己的真实信息。
